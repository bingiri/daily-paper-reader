## 摘要
最近统一的多模态模型表明，单一架构可以同时执行视觉/语言理解和图像生成/编辑。然而，它们会反复将所有历史视觉和文本输入馈送到共享上下文窗口中，由于视觉令牌爆炸和不可靠的跨轮次引用，限制了长程多模态对话。我们提出了一种认知结构化多模态智能体，它将视觉信息外部化到情节视觉记忆中，并在推理过程中选择性地重新激活相关情节。该智能体由用于结构化视觉抽象的感知抽象引擎、用于跨轮次记忆检索的认知检索引擎以及用于自主任务推断和行动规划的多模态执行控制器组成。为了解决现有数据集中缺乏轮次级别检索监督的问题，我们开发了一个统一场景引擎，通过程序生成具有细粒度检索注释的结构化多轮对话，使得强化学习能够优化抽象和检索策略。我们还构建了一个按难度分层的长程视觉对话基准，以评估情节视觉回忆。我们的8B智能体在20轮对话中实现了91.4%的检索准确率，超过了32B基线模型8.2%，同时几乎将每轮推理时间减半（从23.1秒降至12.7秒）。我们进一步提出了认知结构化多模态智能体工具包（CMA-Harness），该智能体部署了相同的认知结构，集成了持久多模态记忆、网络访问、图像生成/编辑/合成工具以及兼容OpenAI的服务。相比于单一的参数扩展，结构化记忆和模块化决策为长程多模态智能体提供了一种更具可扩展性和效率的范式。代码：https://github.com/caseclose/cma-harness；项目页面：https://caseclose.github.io/cma-harness/

## Abstract
Recent unified multimodal models show a single architecture can jointly perform vision/language understanding and image generation/editing. However, they repeatedly feed all historical visual and textual inputs into a shared context window, limiting long-horizon multimodal dialogue due to visual token explosion and unreliable cross-turn referencing. We propose a Cognitive-structured Multimodal Agent that externalizes visual information into an Episodic Visual Memory and selectively reactivates relevant episodes during reasoning. The agent consists of a Perceptual Abstraction Engine for structured visual abstraction, a Cognitive Retrieval Engine for cross-turn memory retrieval, and a Multimodal Executive Controller for autonomous task inference and action planning. To address the lack of turn-level retrieval supervision in existing datasets, we develop a Unified Scenario Engine that programmatically generates structured multi-turn conversations with fine-grained retrieval annotations, enabling reinforcement learning to optimize abstraction and retrieval policies. We also construct a long-horizon visual-dialogue benchmark stratified by difficulty to evaluate episodic visual recall. Our 8B agent achieves 91.4% retrieval accuracy over 20-turn sessions, surpassing 32B baselines by +8.2% while nearly halving per-turn inference time (23.1s -> 12.7s). We further present the Cognitive-structured Multimodal Agent Harness (CMA-Harness), a tool-augmented deployment of the same cognitive structure integrating persistent multimodal memory, web access, image generation/editing/composition tools, and OpenAI-compatible serving. Structured memory and modular decision-making offer a more scalable, efficient paradigm for long-horizon multimodal agents than monolithic parameter scaling. - **研究动机**：现有统一多模态模型（如 BAGEL、Chameleon 等）将视觉和语言理解、图像生成本身集成到单个架构中，但它们在**长程多模态对话**场景下存在两个关键瓶颈：
  - **视觉 Token 爆炸**：每轮对话需要重复输入所有历史图像，视觉 Token 数量随对话轮次线性增长，严重挤占上下文窗口。
  - **跨轮引用不可靠**：依赖隐式注意力机制处理长上下文，导致检索错误和语义漂移，在复杂跨主题、长间隔回调用时表现极差（在 Hard 子集上检索准确率低于 3%）。
- **研究含义**：参数扩展（从 8B 到 32B）无法根本解决上述问题，需要**结构化记忆管理**和**模块化决策**，将视觉信息外部化并选择性激活，从而实现可扩展、高效的长程多模态交互。

## 2. 方法论

### 核心思想
将多模态对话建模为**视觉上下文选择问题**，通过认知结构化分解，使得感知、记忆、检索、执行各模块独立可训练、可替换。

### 关键技术细节

1. **系统架构**（以图 2 为参考）：
   - **感知抽象引擎 (Perceptual Abstraction Engine, PAE)**：将输入图像抽象为结构化记忆条目（标签、描述、缩略图），存入**情景视觉记忆 (Episodic Visual Memory, EVM)**。
   - **认知检索引擎 (Cognitive Retrieval Engine, CoRE)**：根据当前查询和对话状态，从 EVM 中检索最相关的视觉情节，输出图像索引列表。
   - **多模态执行控制器 (Multimodal Executive Controller, MEC)**：进行任务分类（生成/编辑/理解/纯聊天），并协调 PAE、CoRE 以及下游生成或理解模型，最终产生系统输出。

2. **训练流程**（SFT + RL 分阶段优化）：
   - **SFT 训练检索器 (CoRE)**：使用程序构建的带轮次级检索标签的对话数据，训练检索。
   - **DAPO 强化学习优化检索器 (CoRE)**：以难度加权的 Jaccard 相似度作为奖励，优化检索策略。
   - **DAPO 强化学习优化记忆编写器 (PAE)**：冻结 CoRE，以检索准确率作为奖励，优化 PAE 的抽象表示，使其对检索任务更有利。

3. **数据与基准构建**：
   - **统一场景引擎 (Unified Scenario Engine)**：使用 Gemini 模拟用户、多智能体系统生成答案，自动产生带检索标注的 20 轮对话，覆盖 8 个领域、55 个主题。
   - **M2CA-Bench**：包含 100 个会话（2000 个标注轮次），按难度分为 Easy / Medium / Hard / Very Hard 四层，并注入高相似度干扰项和无需检索的负样本。

## 6. 主要结论与发现

1. **结构化记忆显著优于参数扩展**：8B 智能体在 M2CA-Bench 上英文检索准确率达 91.4%，超过 32B 全上下文基线 8.2 个百分点。
2. **检索准确率差距随难度增大**：在 Hard 子集上，本文方法比 8B 多智能体基线高出 18.4 个百分点。
3. **推理时间几乎减半**：每轮推理时间从 23.1 秒降至 12.7 秒（1.8× 加速）。
4. **生成质量提升具有乘数效应**：检索准确率提升 12% 带来生成质量提升约 0.6-1.43 分。
5. **分阶段 SFT+RL 训练至关重要**：仅 SFT 提升 7.3%，RL 进一步累计提升 11.1%。
6. **纯文本记忆不足以应对细粒度视觉检索**：文本记忆在 Hard 子集上仅 24.4%-39.0%，远不如全 EVM（82.0%）。

## 7. 优点

- **设计创新**：将长程对话中的视觉上下文选择作为独立问题，提出认知结构化分解（感知、记忆、检索、执行分离），各模块可独立优化。
- **数据自动生成**：统一场景引擎程序化生成带检索标签的多轮对话，解决了监督数据缺失瓶颈。
- **训练闭环**：通过 RL 将记忆表示优化与下游检索效果直接关联，避免中间指标与最终任务不一致。
- **效率与性能双赢**：8B 模型超越 32B 模型，且推理时长大幅下降。
- **可扩展性**：CMA-Harness 展示了相同认知结构在工具增强、持久记忆、Web 搜索等实际部署中的可行性。
