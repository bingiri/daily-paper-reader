## 摘要
尽管多模态大语言模型（MLLMs）在通用任务中表现出色，但由于单一线性规划的局限性，严格的科学推理仍然具有挑战性。这种顺序设计常常面临视觉-语义错位、长上下文幻觉以及在固定任务粒度下的脆弱执行。我们提出了TopoAgent，一种自演化拓扑框架，用动态、状态隔离的图演化替代线性轨迹。TopoAgent首先采用前端分解器将复杂查询分解为视觉基础的原子。这些原子根据其依赖关系组织成有向无环图（DAG），从而实现严格的上下文隔离，以屏蔽推理引擎免受无关历史噪声的干扰。此外，我们引入了自适应原子分裂，当工具能力边界被超出时，在运行时会动态地将瓶颈节点拆分为更细粒度的子原子。在数学、物理和化学基准上的大量实验表明，TopoAgent显著优于最先进的线性智能体框架，为自主科学推理提供了一种鲁棒、抗噪声且自校正的范式。

## Abstract
While Multimodal Large Language Models (MLLMs) excel in general tasks, rigorous scientific reasoning remains challenging due to the limitations of monolithic, linear planning. Such sequential designs often suffer from visual-semantic misalignment, long-context hallucinations, and brittle execution under fixed task granularity. We propose TopoAgent, a self-evolving topological framework that replaces linear trajectories with dynamic, state-isolated graph evolution. TopoAgent first employs a front-end decomposer to fracture complex queries into visually-grounded atoms. These atoms are organized into a Directed Acyclic Graph (DAG) based on their dependencies, enabling strict context isolation to shield the reasoning engine from irrelevant historical noise. Furthermore, we introduce adaptive atomic fission, which dynamically splits bottleneck nodes into finer-grained sub-atoms at runtime when tool capability boundaries are exceeded. Extensive experiments across mathematics, physics, and chemistry benchmarks demonstrate that TopoAgent significantly outperforms state-of-the-art linear agent frameworks, providing a robust, noise-resistant, and self-correcting paradigm for autonomous scientific reasoning.

### 2. 论文提出的方法论：核心思想、关键技术细节

-   **核心思想**：用**拓扑图演化**替代**线性轨迹累积**。即将复杂问题分解为独立的、视觉基础的原子任务，并以有向无环图（DAG）的形式组织它们，实现状态隔离和动态调整。

-   **关键技术细节**：
    1.  **视觉基础的原子分解（Visually-Grounded Atomic Decomposition）**：
        -   一个前端**分解器**模块将复杂问题分解为一系列最小的、可执行的单元，称为**原子**。
        -   每个原子（Atom）是一个“感知优先”（perception-first）的任务，即先锚定视觉元素并将其转换为确定的符号状态（如“提取物体高度h=3m”），再进行后续逻辑计算。

    2.  **有向无环图规划与上下文隔离（DAG Planning & Context Isolation）**：
        -   这些原子节点根据其内在的依赖关系被组织成一个有向无环图。
        -   执行时，系统引入**上下文隔离**机制：节点`Ai`的输入`C(Ai)`严格限定为其所有直接前驱节点的确定性执行结果`M(Aj)`之和，而**不**是全部历史。公式为：
            ```
            C(Ai) = U_{Aj ∈ P(Ai)} M(Aj)
            ```
            （其中P(Ai)是Ai的前驱节点集）
        -   这种拓扑隔离有效屏蔽了无关噪声，防止了长期推理中的注意力稀释。

    3.  **自适应原子分裂（Adaptive Atomic Fission）**：
        -   为了解决固定任务粒度的缺陷，系统引入**自适应原子分裂**机制。
        -   每个原子执行时都有一个最大步骤限制`Tmax`。如果未能在此限制内达到“STOP”状态，则该节点被标记为**能力超限节点**。
        -   系统触发一个**分裂函数`F`**，由LLM分析失败轨迹，将瓶颈节点`Ai`分解为更细粒度的子原子：
            ```
            F(Ai, trace) → {Ai.1, Ai.2, ..., Ai.k}
            ```
        -   这些子原子被拓扑排序并放入一个动态双端队列`Qexec`的头部，优先解决当前瓶颈。
        -   **分层依赖映射**确保子原子执行完毕后，父节点的结果由子节点结果自动聚合，无需重建全局DAG。

-   **算法流程**（总结）：
    1.  输入问题、图像、工具集和最大步骤`Tmax`。
    2.  分解器生成DAG。
    3.  对DAG进行拓扑排序，压入执行队列。
    4.  循环从队列取出一个原子，构建其隔离上下文`C(Ai)`。
    5.  在当前原子的`Tmax`步内，调用规划器选择工具并执行。
    6.  **成功**：存储结果到内存。
    7.  **失败**：触发分裂函数生成子原子，将子原子压入队列前端，返回步骤4继续。
    8.  所有原子执行完毕后，生成最终答案。
