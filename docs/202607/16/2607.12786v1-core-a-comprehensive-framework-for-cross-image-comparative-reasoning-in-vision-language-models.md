## 摘要
跨图像比较推理对视觉语言模型（VLM）仍然具有挑战性，尤其是在正确预测需要细粒度属性定位和全局一致推理时。我们提出CoRe，一个针对该问题的统一框架。CoRe包括：(i) CoRe-20K，一个通过多专家协作流程从结构化视觉元数据自动构建的大规模三元组训练集，涵盖计数、深度、距离和空间关系；(ii) TriSR，一个结构化奖励框架，在GRPO优化下联合监督属性定位、判断对齐和三重一致性；(iii) CoRe-Bench，首个专用于细粒度跨图像比较推理的基准。实验表明，CoRe在CoRe-Bench上显著优于现有VLM，同时在标准多模态基准上保持竞争力，在部分准确率上比最强基线提升28.2个百分点。

## Abstract
Cross-image comparative reasoning remains challenging for vision-language models (VLMs), especially when correct prediction requires fine-grained attribute grounding and globally consistent reasoning. We present CoRe, a unified framework for this problem. CoRe includes: (i) CoRe-20K, a large-scale triplet-based training set automatically constructed from structured visual metadata through a multi-expert collaborative pipeline, covering counting, depth, distance, and spatial relations; (ii) TriSR, a structured reward framework that jointly supervises attribute grounding, judgment alignment, and triplet consistency under GRPO optimization; and (iii) CoRe-Bench, the first benchmark dedicated to fine-grained cross-image comparative reasoning. Experiments show that CoRe substantially outperforms existing VLMs on CoRe-Bench while remaining competitive on standard multimodal benchmarks, achieving a 28.2-point gain in partial accuracy over the strongest baseline.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：视觉语言模型（VLM）在处理**跨图像细粒度比较推理**时存在系统性不足。现有模型能理解单张图像，或进行多图像语义整合（如时间顺序、叙事理解），但无法准确比较多张图像中涉及的**计数、深度、距离、空间关系**等可量化属性。例如，模型可能得出正确最终答案，但推理链存在事实错误或逻辑不一致。
- **研究背景**：
  - 现有基准（如 MME、BLINK、MuirBench）侧重单图感知或多图语义理解，缺乏对细粒度跨图像比较推理的覆盖。
  - 仅基于最终答案正确性的监督（outcome-only）会导致模型通过捷径或偶然猜测得出正确结果，掩盖推理缺陷；而 LLM-as-Judge 评估在跨图像比较推理中与人类判断存在明显偏差。
  - 因此需要可扩展的数据、可验证的中间监督以及专用评估基准。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- 提出 **CoRe 框架**，包含三项贡献：
  1. **CoRe-20K**：从结构化视觉元数据自动构建的大规模三元组（triplet）训练集（20,000个三元组，对应60,000个比较问题），覆盖计数、深度、距离、空间关系四个任务维度。
  2. **TriSR**：一种**结构化奖励框架**，将推理评估分解为三个可验证的组件：**属性定位（Attribute Grounding）、判断对齐（Judgment Alignment）、三元组一致性（Triplet Consistency）**，并与 GRPO（Group Relative Policy Optimization）结合优化。
  3. **CoRe-Bench**：首个专用跨图像比较推理基准（4000个评估三元组，每个任务1000个）。

### 关键技术细节
- **数据构建（多专家协作流水线）**：
  - **度量提取专家**：从结构化标注（如实例计数、深度图、3D标注、边界框）提取统一度量值。
  - **质量控制专家**：通过噪声裕度过滤和琐碎样本排除，确保三元组中所有成对比较具有足够区分度且非平凡。
  - **问题生成专家**：基于模板库生成成对比较问题（三张图之间三对），随机化选项位置消除偏差。
- **TriSR 结构化奖励**：
  - **属性定位奖励** \(R_{ag}\)：鼓励模型正确估计每张图像的任务相关属性值（如人数），使用指数衰减函数衡量估计值与真值的接近程度。
  - **判断对齐奖励** \(R_{ja}\)：同时检查推理链中的中间判断是否与真值一致、是否与模型自己的最终答案一致。
  - **三元组一致性奖励** \(R_{tc}\)：验证三个成对比较结果是否满足逻辑一致性（如如果 A>B 且 B>C，则 A>C）；对有序比较和区间答案均定义验证器。
  - 复合奖励：\(R = R_{task} + \lambda_{ag} R_{ag} + \lambda_{ja} R_{ja} + \lambda_{tc} R_{tc}\)，其中 \(\lambda_{ag}=0.2, \lambda_{ja}=0.2, \lambda_{tc}=0.3\)。
- **优化**：使用 GRPO，对每个查询采样多个响应，计算组归一化优势，更新策略；同时加入格式奖励、余弦思维链长度惩罚、重复惩罚作为训练稳定器（非本工作贡献）。

- **CoRe 在跨图像比较推理上大幅超越现有 VLM**：在 CoRe-Bench 上总体准确率22.2%（基线最高6.6%），部分准确率54.0%（基线最高35.6%），提升分别达15.6和28.2个百分点。在所有四个任务维度上均有一致改进。
