## 摘要
4D时空推理，即联合建模3D空间结构和时间演化，对于理解动态世界和实现具身交互至关重要。尽管当前的多模态大语言模型（MLLMs）在静态场景理解和粗粒度4D任务中表现出强大能力，但在连续动态场景感知中仍存在显著局限，特别是在跟踪动态对象证据以进行连贯的4D时空推理方面。这一不足主要源于依赖稀疏的帧级观测，将连续动态线索碎片化，使模型无法区分真实的物体动态与相机引起的视在运动。受人类在补偿视点变化的同时跟踪动态线索的启发，我们提出了DynTrace，一个无需训练的4D时空推理框架，包含两个互补组件。动态轨迹可视化（DTV）将世界坐标轨迹重新投影到图像平面，提供基于几何信息的视觉先验，从而区分真实的物体动态与相机引起的视在运动。同时，动态轨迹标记（DT-Token）被组织成动态轨迹图（DTG），跟踪对象级动态线索、轨迹演化及关键时刻，为连贯的4D推理维持连续动态对象证据。这两个组件共同为MLLMs提供持续跟踪的动态对象证据，这些证据基于几何信息的视觉先验和结构化的时空轨迹。DynTrace持续改进开源MLLMs，在Dyn-Bench、VLM4D和DSI-Bench上取得了最先进的结果，验证了跟踪动态对象证据对于稳健的4D时空推理的重要性。

## Abstract
4D spatio-temporal reasoning, jointly modeling 3D spatial structure and temporal evolution, is essential for understanding dynamic worlds and enabling embodied interaction. While current Multimodal Large Language Models (MLLMs) show strong capabilities in static scene understanding and coarse-grained 4D tasks, they still have notable limitations in continuous dynamic scene perception, especially in tracking dynamic object evidence for coherent 4D spatio-temporal reasoning. This shortcoming stems mainly from relying on sparse frame-level observations, fragmenting continuous dynamic cues and leaving models unable to disentangle genuine object dynamics from camera-induced apparent motion. Inspired by humans tracking dynamic cues while compensating for viewpoint changes, we propose DynTrace, a training-free framework for 4D spatio-temporal reasoning with two complementary components. Dynamic Trajectory Visualization (DTV) reprojects world-coordinate trajectories onto the image plane, providing geometry-informed visual priors that disentangle genuine object dynamics from camera-induced apparent motion. Meanwhile, the Dynamic Trace Token (DT-Token), organized into a Dynamic Trace Graph (DTG), tracks object-level dynamic cues, trace evolution, and key moments, maintaining continuous dynamic object evidence for coherent 4D reasoning. Together, these two components equip MLLMs with continuously tracked dynamic object evidence, grounded in geometry-informed visual priors and structured spatio-temporal traces. DynTrace consistently improves open-source MLLMs, achieving state-of-the-art results on Dyn-Bench, VLM4D, and DSI-Bench, validating the importance of tracking dynamic object evidence for robust 4D spatio-temporal reasoning.

### 2. 论文提出的方法论：核心思想、关键技术细节

-   **核心思想**：提出一个名为 **DynTrace** 的**无需训练**的推理框架。该框架通过将原始视频转换为显式的、连续的动态物体证据，来增强现有MLLM的4D推理能力。其核心是耦合了两个互补的组件：**动态轨迹可视化**和**动态轨迹图**。

-   **关键技术细节**：
    -   **框架流程（三个步骤）**：
        1.  **动态物体提取**：首先使用MLLM根据用户问题从视频中提取出相关的视觉实体描述。然后，通过WAFT光流估计和背景运动（相机运动）消除，分离出独立运动的物体区域。最后，利用SAM3进行分割和跟踪，生成与查询相关且时间一致的运动物体掩码。
        2.  **时空动态编码**：此阶段将获取的物体掩码提升到共享的世界坐标系中，重建出“几何基础的动态证据”。
            -   **动态轨迹可视化 (DTV)**：将物体在世界坐标系中的运动轨迹重新投影回图像平面，并用箭头等视觉线索进行标注。这使得MLLM可以直接看到物体的真实运动方向，而不是被相机运动所干扰。
            -   **动态轨迹图 (DTG)**：将时间轴划分为多个窗口，为每个窗口中的每个物体和每对物体创建**动态轨迹标记**。这些标记记录了**动态线索**（位置、速度、朝向）、**轨迹演化**（趋势、加速度）和**关键时刻**（转向、最接近等）。所有的标记和相机行为摘要被组织成一个可查询的图结构。
        3.  **表示整合与推理**：将用户查询、DTV（提供视觉基础）和序列化的DTG（提供结构化文本证据）一起输入到目标MLLM中，最终由MLLM给出答案。

    -   **动态物体提取**：
        -   `E = MLLM(V, q)` : 从视频`V`和查询`q`中提取实体描述`E`。

### 7. 优点：方法或实验设计上的亮点

-   **方法巧妙**：灵感来源于人类认知，将“补偿自身运动”和“追踪物体”这一耦合机制显式化为无需训练的推理框架，设计思路清晰且具有生物学启发性。
-   **无需训练**：这一点是巨大的优势。它可以直接应用于任何现有MLLM，无需重新训练或微调，大大降低了应用门槛和计算成本。
-   **模块化设计**：DTV和DTG两个组件相互独立又协同工作，可以进行灵活的定制和替换。
-   **信息显式化**：将隐式的动态信息（如物体运动、空间关系演化、关键事件）显式地提取并结构化（文本+视觉），使得MLLM不需要自己去“猜测”这些信息，从而降低了推理难度，提高了准确性。
-   **实验设计全面**：不仅在主任务上取得了SOTA，还通过详尽的消融、失败模式分析、鲁棒性测试等，深入剖析了方法的有效性和局限性，论证充分，科学严谨。
-   **对错误的根源分析深入**：首先分析出两种具体的错误模式，然后针对性地设计解决方案（DTV解决源混淆，DTG解决轨迹丢失），体现了精准的问题驱动研究范式。

### 8. 不足与局限

-   **对中间感知误差的鲁棒性**：DynTrace高度依赖SAM3和DA3等模型的预测结果。尽管实验表明其对偶然误差具有一定的鲁棒性，但流程中任何一个中间步骤的严重失败（如物体分割出错、深度估计不准确）都可能导致最终推理错误。
-   **细粒度物体/部件推理局限**：如论文在“当前局限”部分承认，当问题涉及到更**细粒度的物体部件**（如“人手相对于狐狸的位置”）或在**物体自身的第一人称视角下**进行推理时，目前DynTrace以整个物体为单位的建模方式可能不足。
-   **计算开销**：虽然无需训练，但DynTrace的推理流程涉及多个模型（MLLM, 光流, SAM3, DA3）的串联调用，引入了额外的计算和延迟。实时应用可能面临挑战。
-   **非连续轨迹处理**：DynTrace在处理物体**暂时被遮挡**后又出现的情况时，虽然有所改善（见遮挡鲁棒性分析），但对复杂场景下长时间的、多变的重识别仍然可能是个挑战。
-   **领域范围**：方法主要针对以物体为中心的动态场景推理。对于纯粹的“摄像机-背景”的动态，或更抽象的时空关系（如因果、意图），其适用性有待进一步探讨。
