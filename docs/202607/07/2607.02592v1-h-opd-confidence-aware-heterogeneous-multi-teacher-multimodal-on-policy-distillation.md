## 摘要
在线蒸馏（OPD）通过为学生生成的轨迹提供监督，最近成为一种有效的训练后范式。然而，现有的用于多模态推理的OPD方法通常依赖于静态的教师路由，即根据模态或任务类型将每个样本分配给单个教师。这忽略了视觉基础推理和抽象推理可能主导不同的解码步骤，使得单个教师不足以覆盖整个轨迹。为此，本文提出了H-OPD，一种置信度感知的异构多教师在线蒸馏框架，用于多模态推理。通过验证同一推理过程中异构教师的互补性，H-OPD将任务或样本级别的教师路由替换为沿着共享学生轨迹的令牌级别的教师仲裁。H-OPD采用视觉到语言描述转移，使纯文本教师能够访问关键的视觉语义，并使用置信度感知的仲裁机制在每个令牌处动态组合视觉-语言教师和纯文本教师。在11个广泛使用的推理基准上的广泛评估展示了我们方法的优越性能。

## Abstract
On-policy distillation (OPD) has recently emerged as an effective post-training paradigm by providing supervision on student-generated trajectories. However, existing OPD methods for multimodal reasoning usually rely on a static teacher routing, assigning each sample to a single teacher based on modality or task type. This ignores that visual grounding and abstract reasoning may dominate different decoding steps, making a single teacher insufficient for the full trajectory. To this end, H-OPD is proposed as a confidence-aware heterogeneous multi-teacher OPD framework for multimodal reasoning. By verifying the complementarity of heterogeneous teachers in the same reasoning process, H-OPD replaces task or sample level teacher routing with token-level teacher arbitration along the shared student trajectory. H-OPD employs vision-to-language description transfer to enable text-only teachers to access key visual semantics, and uses a confidence-aware arbitration mechanism to dynamically combine vision-language teacher and text-only teachers at each token. Extensive evaluations over 11 widely-used reasoning benchmarks showcase the superior performance of our method.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **多模态推理中的训练后范式瓶颈**：现有的多模态大语言模型（MLLMs）训练后通常采用监督微调（SFT）和强化学习（RL）两阶段范式。SFT易受暴露偏差影响，而RL依赖序列级奖励且训练成本高。在线蒸馏（OPD）作为一种替代范式，通过教师对学生生成的轨迹进行监督，具有高效优势。
- **现有OPD方法的不足**：当前多模态推理的OPD方法采用静态教师路由（task/sample-level routing），即根据模态或任务类型为每个样本分配单一教师。这种粗粒度的分配忽略了同一推理轨迹中不同解码步骤对视觉感知和抽象推理的不同需求——教师可靠性会随令牌状态波动。例如，视觉模型能正确感知但推理出错，而纯文本模型在推理密集型步骤更可靠。
- **核心动机**：本文旨在实现“令牌级教师仲裁”，让视觉-语言教师和纯文本教师在同一推理轨迹中互补协作，动态分配监督权重，从而提升多模态推理效果。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式/算法流程

- **核心思想**：H-OPD 提出**令牌级异构多教师在线蒸馏**框架，将“基于样本的静态路由”转变为“基于令牌状态的动态仲裁”。具体来说，同一输入同时送入视觉-语言教师（VL Teacher）和纯文本教师（Text Teacher），并在每个解码令牌上根据教师置信度动态融合二者的分布作为监督目标。
- **关键技术细节**：
  1. **视觉到语言描述转移（Vision-to-Language Description Transfer）**：为弥补纯文本教师缺乏视觉感知的缺陷，首先利用VL教师为输入图像生成任务相关的文字描述 \(d\)，然后将该描述拼接至原始问题文本中构造增强提示 \(\tilde{x} = x_{txt} \oplus [\text{IMAGE DESCRIPTION}] \oplus d\)，使文本教师可基于文本线索进行推理。此过程离线完成，不增加在线开销。
  2. **置信度感知的令牌级仲裁**：在每个解码步 \(t\)，从两个教师获得截断的top-k分布 \(\bar{q}_V^{(t)}\) 和 \(\bar{q}_T^{(t)}\)，并合并候选词汇集 \(\Omega_t = K_V^{(t)} \cup K_T^{(t)}\)。通过计算每个教师分布的熵作为不确定性度量，得到置信度得分 \(s_V^{(t)} = -H(\bar{q}_V^{(t)}), s_T^{(t)} = -H(\bar{q}_T^{(t)})\)，然后通过softmax生成仲裁权重 \(\lambda_t = \frac{\exp(s_V^{(t)}/\tau_f)}{\exp(s_V^{(t)}/\tau_f) + \exp(s_T^{(t)}/\tau_f)}\)。最终目标分布为 \(q_A^{(t)} = \lambda_t \bar{q}_V^{(t)} + (1-\lambda_t)\bar{q}_T^{(t)}\)。
  3. **优化目标**：学生模型最小化其分布与仲裁目标之间的反向KL散度，仅计算合并支持集 \(\Omega_t\) 上的散度：\(\mathcal{L}(\theta) = \mathbb{E}_{y\sim\pi_\theta}\left[\sum_t m_t D_{\Omega_t}^{\text{KL}}(p_\theta^{(t)} \| q_A^{(t)})\right]\)，其中 \(m_t\) 为有效位置掩码。
- **算法流程**（文字描述）：阶段一离线生成图像描述并过滤；阶段二在线进行学生轨迹采样，对每个解码步并行查询两个教师，截取top-k，计算熵得出置信度，融合得到仲裁目标，计算KL损失更新学生。

### 6. 论文的主要结论与发现

- **H-OPD显著优于现有方法**：在11个benchmark上平均得分提升4.9%–5.5%（相对于单教师OPD/ExOPD），尤其在逻辑推理和图表理解任务中增益明显。

- **Token效率高**：H-OPD相比GRPO在更少的训练步骤和生成令牌中达到更高准确率，收敛更快。
- **可扩展性强**：使用更强教师（RL-trained）或更大的教师/学生模型时，H-OPD持续优于对比方法，表明其鲁棒性和通用性。

### 7. 优点：方法或实验设计上的亮点

- **方法创新**：首次提出令牌级异构多教师在线蒸馏框架，突破了传统任务/样本级静态路由，实现细粒度、动态的监督信号融合。
- **视觉-语言桥接设计**：通过将视觉信息转化为文本描述，使纯文本教师可参与到多模态推理监督中，无需改变教师架构，简单高效。

### 8. 不足与局限

- **依赖描述质量**：纯文本教师的效果受限于视觉描述的质量；若描述不完整、含噪声或与任务对齐差，则监督信号可能次优。论文通过GPT-4.1-mini过滤以减少噪声，但仍有残留风险。
- **置信度-正确性偏差**：低熵≠正确，当教师过于自信或校准不良时，基于熵的仲裁可能信任错误token。
- **训练开销增加**：相比单教师OPD，H-OPD需要并行查询两个教师，虽然合并top-k降低了计算，但每步仍有多教师推理开销。论文通过离线描述生成缓解部分负担。
- **截断信息损失**：top-k截断可能丢弃具有信息量的低概率候选，影响细粒度监督。
- **可扩展性未充分验证**：受限于计算资源，未在更大规模（如MoE或32B级模型）上进行实验；论文承认这是未来工作。
- **实验统计信息不足**：未报告多次运行的标准差或置信区间，无法评估结果的稳定性。
