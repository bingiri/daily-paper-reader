## 摘要
多模态大语言模型（MLLMs）在细粒度视觉推理中常常失败，因为与问题相关的视觉线索被密集冗余的图像令牌稀释。最近的多模态推理方法通常将语言模型中的思维链扩展到视觉或潜在空间，试图添加中间推理状态，同时忽略了冗余视觉令牌的负面影响。我们提出潜在噪声掩码（Lens），一种基于问题的视觉证据净化框架，使MLLMs能够在潜在空间中使用更干净的视觉线索进行推理。Lens引入轻量级的Lens证据令牌（LET）来评估哪些视觉令牌支持当前问题，并在解码过程中保留它们。在LET得分的引导下，它向低相关性令牌注入自适应的潜在噪声，在不改变模型主干或令牌序列的情况下软性地抑制干扰项。仅使用一个临时可学习的控制令牌和一个轻量级噪声生成器，Lens在大多数VQA数据集上提升了基础MLLM 2.4-6.4个点，在定位任务上提升了4.1-6.4个点，同时增加的开销极小。这些结果表明，多模态推理可以从更干净的与问题相关的视觉证据中获益，而非简单地扩展推理轨迹。

## Abstract
Multimodal large language models (MLLMs) often fail in fine-grained visual reasoning, as question-relevant visual cues are diluted by dense and redundant image tokens. Recent multimodal reasoning methods usually extend chain-of-thought from language models into visual or latent spaces, seeking to add intermediate reasoning states while overlooking the negative impact of redundant visual tokens. We propose LatEnt Noise maSk (Lens), a question-conditioned visual evidence purification framework that empowers MLLMs to reason with cleaner visual cues in latent space. Lens introduces a lightweight Lens Evidence Token (LET) to score which visual tokens support the current question and preserve them during decoding. Guided by the LET scores, it injects adaptive latent noise into low-relevance tokens, softly suppressing distractors without changing the model backbone or token sequence. With only one temporary learnable control token and a lightweight noise generator, Lens adds minimal overhead while improving the base MLLM by 2.4-6.4 points on most VQA datasets and by 4.1-6.4 points on grounding tasks. These results show that multimodal reasoning can benefit more directly from cleaner question-relevant visual evidence than from simply extending the reasoning trace.

## 2. 论文提出的方法论

- **核心思想**：提出 **LatEnt Noise maSk（Lens）** ——一个基于问题的视觉证据净化框架。它通过引入一个临时可学习的 **Lens Evidence Token（LET）**，评估每个视觉 token 与当前问题的相关性，然后根据得分向低相关 token 注入自适应潜在噪声，软性地抑制干扰，同时保持原始 token 序列结构和模型主干不变。
- **关键技术细节**：
  1. **证据探测（Evidence Probing）**：在输入序列 [V, T] 后附加一个临时 token `<mask>`，通过 MLLM 解码器得到其隐藏状态，再由一个单层 MLP 头映射为 LET 得分向量 **a** ∈ [0,1]^N，表示每个视觉 token 支持问题的概率。使用物体级边界框监督：若 token 对应的图像块与问题相关的标注框有重叠，则标记为正样本，否则为负样本。损失为二元交叉熵。
  2. **潜在抑制（Latent Suppression）**：根据 LET 得分和阈值 τ，计算每个 token 的门控值 g_i = (1/τ) * ReLU(τ - a_i)。一个轻量噪声生成器 G_phi 为每个视觉 token 预测均值 μ_i 和正标准差 σ_i，并利用重参数化采样得到潜在噪声 r_i，然后对低得分 token 施加扰动：ev_i = v_i + g_i * r_i。高得分 token 保持不变。
  3. **优化**：先进行监督微调（SFT），联合优化答案损失和 LET 损失（权重 β=0.2）。之后进行强化微调（GRPO），将 LET 得分建模为伯努利策略，以证据集的 F1 分数作为奖励，并加入 KL 散度正则化项，保持策略接近 SFT 后的分布。
- **公式/算法流程**（文字说明）：
  - 输入图像 I，问题 Q → 图像编码器 → 视觉 token V，文本 token T。
  - 拼接 V、T、临时 token → MLLM 解码 → 从临时 token 隐状态预测 LET 得分 a。
  - 根据 a 和 τ 计算门控 g，噪声生成器产生 ri，构建净化后视觉 token eV。
  - 移除临时 token，用 eV 替换原始视觉 token，进行标准自回归解码生成答案。

## 6. 论文的主要结论与发现

- **核心发现**：通过净化视觉证据（即抑制与问题无关的 token）比简单扩展推理轨迹（增加 CoT 或潜在思维 token）更能直接有效地提升 MLLM 的细粒度推理。
- **具体性能提升**：Lens-SFT 使平均得分从 SFT 的 66.40 提升至 70.99；Lens-GRPO 进一步提升至 75.44，尤其是接地任务提升显著（如 VisDrone 提升 18.81 点，COCO 提升 14.43 点），证明视觉选择性的直接益处。
- **泛化性**：在从 2B 到 9B 的 9 个基座模型上稳定提升 3.0–9.0 个点（平均），验证了方法无关模型家族和规模。
- **组件重要性**：证据探测与潜在抑制必须耦合使用，单独任一效果有限。基于 LET 得分的软性噪声抑制显著优于随机掩码、零填充或基于注意力分数的掩码。
- **计算效率**：Lens 仅增加一次探测前向和 token 级修正成本，解码速度几乎不变，平均回答长度与 SFT 一致（约 14-16 token），说明性能提升并非来自更长的输出。

## 7. 优点

- **方法轻量且通用**：仅引入一个临时控制 token 和一个轻量噪声生成器，不改变模型主干、不增加输出长度、不依赖外部工具（检测器、生成器等），易于附加到现有 MLLM 上。
- **软性抑制，鲁棒性强**：通过潜在噪声而非硬删除或剪枝token，保留了原始序列结构，使模型在相关性预测不完美时仍可利用部分信息，避免了硬性取舍的脆弱性。
- **监督成本低**：利用物体级边界框（而非像素级掩模或交心理性）即可生成 token 级标签，易于从现有标注数据中获得。
- **强化微调策略**：将 LET 得分建模为伯努利策略，以集合级 F1 作为奖励进行优化，更符合选择完整证据集的目标，优于独立 token 分类或仅优化答案的强化方法。
- **实验严谨**：在多项基线上公平对比，消融充分，并提供了跨模型泛化、超参敏感性、推理成本等多维度分析。

## 8. 不足与局限

- **对边界框标注的依赖**：证据探测需要问题相关的物体框标签作为训练监督。在某些场景下（如抽象概念推理、开放式问答），这种标注难以获取，限制了方法的扩展性。
- **阈值敏感性**：虽然 hyperparameter 消融显示τ=0.5和0.7差异不大，但不同 τ 对 VQA 和接地任务的影响存在 trade-off，通用最优阈值可能因任务而异。
- **噪声注入的风险**：对于极端小目标、高度遮挡或混淆物，噪声抑制可能意外削弱有用信息。尽管可视化显示效果良好，但缺乏更严格的理论保证。
- **模型覆盖不足**：仅在 Qwen3-VL 和 InternVL3.5 两个系列上测试，未在 LLaVA-NeXT、CogVLM 等其他主流 MLLM 上验证通用性。
- **任务覆盖有限**：评估集中在视觉问答和物体定位上，未涉及文档理解、数学推理、多轮对话等更复杂的多模态推理场景。
- **训练成本**：虽然推理开销小，但 SFT + GRPO 两阶段训练仍需额外计算资源（尽管与对比方法相当），且强化阶段可能需要调整 KL 正则化参数以避免策略崩塌。
